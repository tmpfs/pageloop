Welcome to your new web page created from the pure template.

About

The pure template is ideal for sketching layouts or single page brochure websites to promote an idea, project or service.

It's intentionally very simple just a blend of quality CSS and HTML with a touch of Javascript for the smooth scroll.

The philosophy is lean. No custom fonts, fancy animations not even any images or SVG, absolute basics!

There is no layout file or markdown in this template if you need those features please choose another template.

Getting Started

You can use the visual (or source code) editor to change your new page, we recommend these steps to get started.

Page Meta

Update the page meta data using the data editor or directly in the index.html file.

Favicon

Content

Express yourself, update the headings and text for your project.

Style
