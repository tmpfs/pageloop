About

This template is ideal for single page brochure websites to promote an idea, project or service.

It's intentionally very simple just a blend of quality CSS and HTML with a touch of Javascript for the smooth scroll.

The philosophy is lean. No custom fonts, fancy animations not even any images or SVG, absolute basics!

There is no layout file or markdown in this template if you need those features please choose another template.

Getting Started

Start to customize your new web page by:

  • Updating the page meta data with a good title, description and keywords.
  • Uploading a favicon.png file.
  • Changing the text content.
  • Updating the app.css stylesheet.
  • Customizing the scripts in app.js.
