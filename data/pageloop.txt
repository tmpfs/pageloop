Usage: pageloop [-h] [--help] [--version] [--addr=<val>] [--config=<file>]
                [--publish=<dir>]

  Collaborative realtime server.

  Web server that serves system applications and
  those specified in the supplied configuration file.

Options
  -a, --addr=[val]        Set the bind address (default: :3577)
  -c, --config=[file]     Load server configuration from a YAML file
  -p, --publish=[dir]     Directory for application builds (default: public)
  -h, --help              Display help and exit
  --version               Print the version and exit
